# Markdown Report Template

Use this layout for every saved report.

```md
# <title>

- 时间: <YYYY-MM-DD HH:mm:ss>
- 原始请求摘要: <1-3 sentence summary>

## 问题分析

<analysis>

## 结论

<conclusion>

## 分阶段执行方案

### 阶段 1: <goal>

- 动作:
- 预期结果:

### 阶段 2: <goal>

- 动作:
- 预期结果:
```

Guidelines:

- Add more phases when needed.
- Keep the plan actionable, not abstract.
- Mention assumptions if they materially affect execution.
