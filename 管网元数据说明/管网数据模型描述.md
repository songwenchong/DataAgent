### 管网数据模型描述
#### 空间元数据表

- 表名称：M_DBMETA
- 描述：用于存储数据库中所有空间数据的表的元数据，包括空间数据的图层名称、空间数据类型、空间数据的 code 等
- 重点字段说明：

| 字段名称  | 含义                                   | 备注                                                         |
| --------- | -------------------------------------- | ------------------------------------------------------------ |
| layername | 空间图层的图层名称，即空间数据的表名称 | 特别说明，type = 4 是管网，此时这个 layername 是管网名称的前缀，其他类型，这个均指表的名称 |
| describe  | 空间图层的图层描述                     | 语义话查找时，查询此管网，如查询供水管网的数据时，根据这个字段找 layername |
| code      | 空间图层的 code                        |                                                              |
| type      | 空间图层的类型                         | 4：管网空间数据；<br/>0：空间线数据；<br/>1：空间点数据<br/>2：空间面数据 |
| layerid   | 空间图层的序号                         | 管网的 layername 对应的 layerid 固定为 -1，具体每个设备的 layerid 在管网的 M_MT 中查看 |

#### 管网模型

##### 管线表

- 表名称：{layername}_lin

- 描述：管网的线表，存储管线空间数据
- 重点字段说明

| 字段名称 | 含义                   | 备注                                                        |
| -------- | ---------------------- | ----------------------------------------------------------- |
| gid      | 表主键                 | 唯一                                                        |
| dno      | 管线设备的设备类型号   | 与 M_MT 表的 dno 对应，设备名称从 M_MT 表对应的 dalias 获取 |
| stnod    | 管线设备的起点的点 gid | 对应 nod 表的 gid                                           |
| ednod    | 管线设备的终点的点 gid | 对应 nod 表的 gid                                           |
| geom     | 管线的空间坐标         |                                                             |
| crttime  | 管线的录入/创建时间    |                                                             |
|          |                        |                                                             |

##### 管点表

- 表名称：{layername}_nod

- 描述：管网的点表，存储管点空间数据
- 重点字段说明

| 字段名称 | 含义                 | 备注                                                        |
| -------- | -------------------- | ----------------------------------------------------------- |
| gid      | 表主键               | 唯一                                                        |
| dno      | 管点设备的设备类型号 | 与 M_MT 表的 dno 对应，设备名称从 M_MT 表对应的 dalias 获取 |
| geom     | 管点的空间坐标       |                                                             |
| crttime  | 管点的录入/创建时间  |                                                             |

##### 管网图层元数据表

- 表名称：{layername}_M_MT

- 描述：管网的图层元数据定义表，定义了这个管网下有几个 dno，每个 dno 对应的 dname 等
- 重点字段说明

| 字段名称 | 含义                               | 备注                                                         |
| -------- | ---------------------------------- | ------------------------------------------------------------ |
| dno      | 管网设备的设备类型号               | 在一个管网下唯一                                             |
| dname    | 管网设备的设备名称                 |                                                              |
| dalias   | 管网设备的设备别名                 | 通常与 dname 一致，但是 dname 可能是英文，所以通常取 dalias 作为页面展示和语义化识别 |
| layerid  | 管网设备在整个空间图层中的图层序号 |                                                              |

##### 管网数据字典表

- 表名称：{layername}_M_MT_FLD

- 描述：管网的点表，存储管点空间数据
- 重点字段说明

| 字段名称 | 含义                 | 备注                                                         |
| -------- | -------------------- | ------------------------------------------------------------ |
| dno      | 管网设备的设备类型号 | 在一个管网下唯一                                             |
| dname    | 管网设备的设备名称   |                                                              |
| name     | 数据库字段名称       |                                                              |
| alias    | 数据库字段的别名     | 一般用显示使用                                               |
| prop     | 字段的特性           | 参考下方的字段特性对照表，字段特性是固定的，每个管网中的字段可自由定义哪些字段是管长特性，哪些字段是管径特性<br/>**在管径/管材/管长等关键字段查询中，需要先查询字段特性，找到对应的字段，然后再以该字段去查询数据** |
